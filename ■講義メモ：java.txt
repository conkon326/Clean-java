javaのプログラムの構造

　大括りにクラスがあって、その中に
　フィールド、コンストラクタ、メソッドを配置する。


javaのデータ型は大きく２つに分けられる。

　・基本データ型(プリミティブ型)
　　byte,short,int,long,float,double,boolean,char の８種類。
　　-------------------                      ----
　　整数型              ------------         文字(1文字)
　　                    小数型       -------
　　                                 真偽(true/false)

　・クラス型(参照型)
　　String など､たくさん。
